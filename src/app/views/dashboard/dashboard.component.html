SAAFA Monitor Web Project


El proyecto consiste en realizar el análisis y evaluación técnica de la placa esp32 para determinar la factibilidad para realizar el monitoreo de las variables definidas en el alimentador automatizado flotante con movilidad controlada con alimentación fotovoltaica autónoma diseñado y construido para dosificar el alimento para el cultivo de tilapia en estanques de geomembrana del área de acuicultura del Instituto Tecnológico de Salina Cruz.

Objetivo

Analizar y evaluar la placa ESP32 para ser usada en el monitoreo y control del alimentador automatizado flotante para el cultivo de tilapia con alimentación fotovoltaica autónoma en el área de acuicultura del Instituto Tecnológico de Salina Cruz.

Importancia de la implementación del Monitor Web en este proyecto

Permitirá la visualización de los parámetros más importantes del sistema de alimentación utilizando la placa SoC ESP32.
